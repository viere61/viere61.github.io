HTML Basics

HTML Basics

Skeleton Code

There is certain code that should be in every HTML document to establish a basic structure and let the browser know some important information about the page.

There are 4 components to an HTML file:

  • <!DOCTYPE html> Declare the document as an HTML file
  • <html> Container for all html
  • <head> Contains all the metadata for the page (For more about metadata: Metadata in HTML)
  • <body> Where all the markup that is to display on the page shows up

If you are using Visual Studion Code:

  • Type <html:5>, then you will automatically get the skeleton code.

Tagging

Generalized Pattern:

  • <tag_name>...<tag_name>

This pattern is mostly true, but there are some exceptions.

Heading:

  • <h1>...<h6>

Paragraph:

  • <p>

Lists:

  • Ordered: <ol>
  • Unordered: <ul>
  • List elements: <li>

Divs:

  • <div>

The HTML division tag, called “div” for short, is a special element that lets you group similar sets of content together on a web page. You can use it as a generic container for associating similar content. It has no effect on the content or layout until styled in some way using CSS.

Image:

  • <img src="" alt="">

src means source, so you put (relative) file path or link to the image.

alt means alternative texts, so if the src goes wrong, alt will appear on the page.

Link:

  • <a href="">this is a link</a>

If you want to open a new tab when you click your link:

  • <a href="" target="_blank">this is a link</a>

Button:

  • <button onclick="">Click Here</button>

Button encourages user to click it.

The onclick event executes a certain functionality when a button is clicked.

Input:

  • <input type="" />

The <input> HTML element is used to create interactive controls for web-based forms in order to accept data from the user; a wide variety of types of input data and control widgets are available.

We use “type” attribute to specify type of inputs.
